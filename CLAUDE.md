## Project Overview

This is a **FortiGate Enterprise Dashboard** - a FastAPI-based web application that provides real-time network topology visualization and management for Fortinet infrastructure. It features:

- Interactive Security Fabric topology visualization (2D and 3D views)
- FortiGate firewall and FortiSwitch management
- Real-time device discovery with manufacturer identification
- Glass-morphism UI with enterprise-grade design
- Power Automate integration for workflow automation
- Comprehensive OUI lookup system with rate limiting and caching
